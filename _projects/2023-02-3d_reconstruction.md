## Background

This project aims to develop a client/server software that can **generate metaHumans from images taken by handheld devices**. The software uses [AliceVision](https://alicevision.org/), a photogrammetric computer vision framework, and [Meshroom](https://github.com/alicevision/Meshroom), a 3D reconstruction software based on AliceVision, to stitch the images together and create a 3D model. The software also allows users to view and export the MetaHuman model, and integrate it with [Unreal Engine](https://www.unrealengine.com/en-US) for advanced applications. The project demonstrates the reconstruction performance, software architecture design, and future work of the software system.
