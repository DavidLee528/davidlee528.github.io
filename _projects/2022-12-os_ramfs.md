## Background

[File systems](https://en.wikipedia.org/wiki/File_system) are essential components of [operating systems](https://en.wikipedia.org/wiki/Operating_system), as they provide a way to store and organize data on persistent storage devices. However, designing and implementing a file system is not a trivial task, as it involves many challenges and trade-offs, such as data structures, algorithms, performance, reliability, security, and portability. In this project, we present the **design and implementation of a simple file system that supports basic file operations, such as creating, deleting, reading, and writing files**. We also describe the internal structure and organization, such as the virtual disk layer, the disk block read/write layer, the disk space allocation and recovery layer, the file management layer, and the terminal application layer. We use `C++` as the programming language and `GNU/Linux`(`Ubuntu 20.04`) as the development environment. We also use some tools and libraries to assist our development and testing process. Our goal is to **demonstrate the main concepts and techniques of file system design and implementation**, and **to provide a simple and clear example for learning purposes**.
